# საქორწილო მოწვევის ვებსაიტი

ერთგვერდიანი (single-page) ელეგანტური მოწვევის საიტი კლასიკური სტილით — თეთრი ფონი, ოქროსფერი აქცენტები, დეკორატიული სვეტები და ყვავილოვანი ილუსტრაციები. ორენოვანი (ქართული + ინგლისური) ენის გადამრთველით.

## დიზაინის მიმართულება

- **ფერები**: ნაზი ოქროსფერი (#C9A961 / #B8924A), ნაზი ბეჟი/კრემისფერი ფონი, მუქი ოქროსფერი ტექსტისთვის, თეთრი ძირითად ფონად
- **ტიპოგრაფია**: 
  - სათაურებისთვის — Cormorant Garamond ან Playfair Display (კლასიკური serif)
  - ძირითადი ტექსტი — Cormorant / Lora
  - ქართული — Noto Serif Georgian (იმავე ელეგანტურ შეგრძნებას ქმნის)
- **ვიზუალური ელემენტები**: 
  - ბერძნული/რომანული სვეტების ილუსტრაციები გვერდის კიდეებზე (hero სექციაში)
  - ყვავილოვანი ორნამენტები (peony, rose, eucalyptus) კუთხეებსა და სექციათა გამყოფებად
  - ოქროსფერი წვრილი ხაზოვანი დეკორი (divider-ები)
- **ანიმაცია**: ნაზი fade-in scroll-ზე (framer-motion), ყვავილების ნაზი parallax

## სექციები (single-page, smooth scroll)

1. **Hero** — სვეტებით ჩარჩოიანი კომპოზიცია, ცენტრში წყვილის სახელები დიდი serif შრიფტით, ქორწილის თარიღი ოქროსფერი ხაზებით, ყვავილოვანი გვირგვინი ზემოთ
2. **Countdown / "We're getting married"** — ნაზი ციტატა, თარიღამდე დარჩენილი დღეების ტაიმერი ოქროსფერი ციფრებით
3. **Our Story** (არასავალდებულო, მოკლე) — წყვილის შესახებ ერთი აბზაცი
4. **Event Schedule / დღის წესრიგი** — vertical timeline ოქროსფერი წერტილებითა და ყვავილოვანი დეკორით (ცერემონია, ფოტოსესია, მისაღები, წვეულება)
5. **Venue & Map / ადგილმდებარეობა** — ვენიუს სახელი, მისამართი, ჩაშენებული Google Maps iframe ოქროსფერ ჩარჩოში, "Get Directions" ღილაკი
6. **Gallery / გალერეა** — წყვილის ფოტოების grid (placeholder ფოტოებით), lightbox-ით (კლიკზე გადიდება)
7. **Footer** — "With love" ხელნაწერი წარწერა, წყვილის ინიციალები მონოგრამად, ყვავილოვანი დეკორი

## ენის გადართვა

- მარჯვენა ზედა კუთხეში პატარა toggle: **ქარ / ENG**
- მარტივი React context, რომელიც ცვლის ყველა ტექსტს ლექსიკონის ფაილიდან (`src/i18n/translations.ts`)
- localStorage-ში ინახება მომხმარებლის არჩევანი
- default — ქართული

## ტექნიკური დეტალები

- **ფაილური სტრუქტურა**:
  - `src/pages/Index.tsx` — მთავარი გვერდის კომპოზიცია
  - `src/components/wedding/Hero.tsx`, `Countdown.tsx`, `Schedule.tsx`, `Venue.tsx`, `Gallery.tsx`, `Footer.tsx`, `LanguageToggle.tsx`
  - `src/components/wedding/decor/` — `Column.tsx` (SVG სვეტი), `FloralCorner.tsx`, `Divider.tsx` (SVG ყვავილოვანი ორნამენტები)
  - `src/i18n/translations.ts` — ტექსტების ლექსიკონი (ka/en)
  - `src/contexts/LanguageContext.tsx` + `useT()` hook
- **დიზაინ-სისტემა** (`src/index.css` + `tailwind.config.ts`):
  - HSL ცვლადები: `--gold`, `--gold-soft`, `--gold-deep`, `--ivory`, `--cream`, `--ink` (მუქი ფონისთვის ტექსტი)
  - გრადიენტები: `--gradient-gold` (ოქროსფერი ტონების გადასვლა divider-ებისთვის)
  - სათაურის შრიფტი ცვლადად, რომ ცვლა ერთ ადგილას ხდებოდეს
  - **არანაირი hardcoded ფერი კომპონენტებში** — ყველაფერი semantic token-ებით
- **ფონტები**: Google Fonts-დან (Cormorant Garamond + Noto Serif Georgian) `index.html`-ში preconnect-ით
- **ყვავილები/სვეტები**: AI-გენერირებული PNG-ები გამჭვირვალე ფონით (`src/assets/`-ში), რომ მსუბუქი იყოს და მკვეთრი დარჩეს ნებისმიერ ზომაზე
- **ანიმაციები**: `framer-motion` (`whileInView` fade/slide-up)
- **გალერეა**: უკვე არსებული `src/components/ui/dialog.tsx` lightbox-ისთვის
- **SEO**: `<title>` წყვილის სახელებით, meta description, Open Graph image (hero-ის სკრინი)
- **რესპონსიული**: მობაილზე სვეტები ნაკლებად შესამჩნევი, ყვავილები მცირდება, schedule timeline ერთსვეტიანი

## პლეისჰოლდერები რომლებიც მერე უნდა შეცვალოთ

- წყვილის სახელები (default: "ანა & გიორგი / Anna & George")
- ქორწილის თარიღი და დრო
- ვენიუს სახელი + მისამართი + Google Maps embed link
- დღის წესრიგის დროები
- გალერეის ფოტოები (default: 6 placeholder)
